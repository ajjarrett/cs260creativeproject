Training Module 1: Scientific Literature




One of the key tenets of biology is that structure and function of biological molecules are intimately related.

When investigating biological molecules, it is very important to get an idea of their structure.

The most common format for storing structural information about biological molecules is the PDB file, which stands for Protein Data Bank format. Structures of biological molecules that have been determined using experimental techniques such as x-ray crystallography or NMR. Each structure has a unique 4-digit code assigned to it.

The worldwide protein data bank database at https://www.rcsb.org/ is a valuable resource that contains structural information for over 185,000 biological molecules.

Use the search box below to view important information about any pdb entry. (If you don't know where to start, try a few of the following pdb codes: 4M7T, 1EDE, 3VNK, 1ALK, 1N68, 1A4L, 5XH3, 4WFI, etc. )
